# Repository Structure

Project submissions are stored under standardized batch and semester folders. Students select visible academic values; the platform creates repository-safe folder names.

```text
batch-2027/
|-- semester-1/
|-- semester-2/
|-- semester-3/
|-- semester-4/
|-- semester-5/
|-- semester-6/
|-- semester-7/
`-- semester-8/
```

Full project paths use this pattern:

```text
batches/{batch}/{semester}/{subject}/{team-number}-{project-name}/
```

Example:

```text
batches/batch-2027/semester-6/dbms/team-01-library-management/
```

The `batch` and `semester` path segments are generated by the platform. Subject and project title values are converted into lowercase URL-safe slugs before upload.
